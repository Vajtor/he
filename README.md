# he
A compiled, bytecode-driven programming language implemented from scratch in pure C.
The core philosophy of **HE** is to explore high-performance language design while targeting a unique, specialized niche. The language is currently under active development and is being built from the ground up using a custom virtual machine architecture.

How to Run & Build

Because the language is in its foundational stages, running the project executes the internal memory verification test suite.

### Prerequisites
* **Windows Users:** Ensure you have MSYS64 installed with `gcc`/`mingw` configured in your system environment path.
* **Editor:** Visual Studio Code.

### Running in VS Code:
1. Open Visual Studio Code and install the **C/C++ Runner** extension (by franneck94 / Intellisense tools).
2. Open this project directory as your active workspace folder.
3. Click the **Run** button provided by the extension in your status bar or press `Ctrl+Alt+N` (or `Cmd+Alt+N` on Mac) to compile and run.
4. **Expected Output:** A correct execution will output an exit code/return value of `0`.
